Данные по курсам валют берутся с сайта ЦБ (http://www.cbr.ru/scripts/XML_daily.asp) по состоянию на текущую дату
Скрипт работает с кешированием результатов. Кеш сохраняется в папке cache. Даже если сервер недоступен, курсы будут браться из кеша.
Имеется две версии калькулятора с разными логотипами: promokey.html и imperia.html
Для вставки информера на сайт достаточно вставить в нужное место код, содержащийся между тегами <body> и </body> соответствующего файла.
Все пути к скриптам и стилям являются относительными и требуют корректировки при установке в подкаталоги сайта.
Вёрстка выполнена в масштабе 1:1 с макета.

Установка:
1. Скопируйте все файлы в корень сайта
2. Убедитесь, что папка cache доступна для записи.
3. В файле js/script.js установите размер наценки в переменной rateTax (по умолчанию установлено 5% - 1,05)
4. Для вставки информера на сайт достаточно вставить в нужное место код, содержащийся между тегами <body> и </body> соответствующего файла.
